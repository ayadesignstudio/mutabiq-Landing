# Mutabiq Landing

Public marketing site for **Mutabiq Cloud** — Saudi DGA-DXMI 2026 compliance auditing platform.

Built with Next.js 15 (App Router) + next-intl for Arabic-first RTL with English locale.

## Develop

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000` — redirects to `/ar` by default.

## Deploy

Push to `main` → Vercel auto-deploys.
